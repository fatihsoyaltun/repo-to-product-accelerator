# Working Agreement

## Scope
Bu ürün genel amaçlı coding agent değildir.
Dar odak: repo havuzundan ürün üretme hızlandırıcısı.

## Human Approval Gates
1. Repo import sonrası: repo listesi doğru mu?
2. Idea matching sonrası: hangi ürün fikri seçildi?
3. Architecture sonrası: MVP kapsamı onaylandı mı?
4. Scaffold sonrası: üretilen yapı kabul edildi mi?

## Rules
- Önce analiz, sonra ürün eşleme, sonra mimari, sonra backlog, sonra scaffold.
- Her aşama dosyaya yazılmalı.
- Tutarsız agent çıktıları QA katmanında işaretlenmeli.
